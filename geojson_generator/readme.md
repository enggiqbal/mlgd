The module generates    `cluster.geojson`,    `cluster_boundary.geojson`,     `edges.geojson`,   `nodes.geojson`,  and   `alledges.geojson` from `map.svg`.

update input map path and output directory then run 

```
$ python3  svg_to_geojson.py
```
